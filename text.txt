🔴 1️⃣ APIهای مدیریت StepFlow (فقط SUPER_ADMIN)
ساخت روش مرحله‌ای
POST /admin/step-flows
ویرایش کامل روش (تغییر عنوان، ترتیب، required، اضافه/حذف فرم)
PUT /admin/step-flows/:id
حذف روش
DELETE /admin/step-flows/:id



لیست همه روش‌های مرحله‌ای
GET /admin/step-flows

🟢 2️⃣ API نمایش برای UI (کاربر عادی)
گرفتن هر دو مود جدا از هم

(فرم‌های تکی + روش‌های مرحله‌ای)

GET /analysis/modes

خروجی:

singleForms (از AnalysisForm)

stepFlows (از StepFlow)

🟣 4️⃣ API حالت تک‌فرمی (همونی که از قبل داری)

گرفتن لیست فرم‌های فعال
GET /analysis/forms
گرفتن جزئیات یک فرم
GET /analysis/forms/:id
ثبت پاسخ فرم تکی
POST /analysis/forms/:id/submit






🔵 API اجرای StepFlow توسط کاربر
شروع یک روش مرحله‌ای

(ساخت StepSession)

POST /step-sessions/start/:flowId
گرفتن سشن فعال کاربر
GET /step-sessions/me
رفتن به مرحله بعد

(با بررسی required)

POST /step-sessions/:id/next
رفتن به مرحله قبل
POST /step-sessions/:id/previous
ثبت پاسخ یک مرحله
POST /step-sessions/:id/submit
انصراف از مسیر
POST /step-sessions/:id/cancel
اتمام کامل مسیر
POST /step-sessions/:id/complete
